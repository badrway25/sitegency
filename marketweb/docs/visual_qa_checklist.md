# Visual QA Checklist — Per Template

## A. Structure
- [ ] Full page scroll reviewed (top to bottom)
- [ ] All sections present and visible
- [ ] No broken empty blocks or excessive whitespace
- [ ] Footer complete and readable

## B. Navbar
- [ ] Desktop navbar visible with all section links
- [ ] Mobile navbar functional
- [ ] Links navigate correctly (no 404)
- [ ] No missing items after pipeline modifications
- [ ] Dropdowns work if present
- [ ] Anchor navigation works
- [ ] Sticky behavior acceptable

## C. Visual Quality
- [ ] Layout feels balanced and intentional
- [ ] Spacing is coherent between sections
- [ ] Typography feels premium (hierarchy, weight, sizing)
- [ ] Buttons are aligned and properly padded
- [ ] Cards are consistent (height, image ratio, spacing)
- [ ] No awkward or low-end section remains
- [ ] Section transitions feel intentional

## D. Images
- [ ] Original branding imagery removed where required
- [ ] Images are premium-quality (Unsplash)
- [ ] Image proportions are correct (no stretch, no crop issues)
- [ ] Card images are visually coherent across a grid
- [ ] No visibly weak or mismatched image

## E. Branding
- [ ] Original template brand name removed
- [ ] New brand name applied consistently (header, footer, hero, title)
- [ ] SVG logo generated and inserted
- [ ] No old brand traces in visible text

## F. Languages (verify each separately)
### IT
- [ ] Full page reviewed in Italian
- [ ] No leftover English text in nav/buttons/CTAs
- [ ] No broken truncation from longer translations

### FR
- [ ] Full page reviewed in French
- [ ] No leftover English text
- [ ] No broken truncation

### EN
- [ ] Full page reviewed in English
- [ ] Content coherent and readable

### AR
- [ ] Full page reviewed in Arabic
- [ ] RTL layout correct
- [ ] No leftover LTR-only text

## G. Arabic / RTL
- [ ] dir="rtl" applied
- [ ] Text alignment correct
- [ ] Card layouts mirrored correctly
- [ ] Navbar/dropdowns usable in RTL
- [ ] Forms aligned correctly
- [ ] No broken section caused by RTL

## H. Preview / Customizer
- [ ] Language sync works between locale and preview content
- [ ] Direction sync works for RTL
- [ ] No stale text after language switch

## I. JS / Runtime
- [ ] No blocking console errors
- [ ] Scroll animations work
- [ ] Counter animations work (no NaN)
- [ ] Back-to-top button works

## J. Premium Judgment
- [ ] Visually elegant — would pass as a real commercial site
- [ ] Clearly above default template quality
- [ ] Not merely functional — genuinely premium
